# Version Control — C++20

## Commits (Conventional)

- `feat(users): add pagination to list handler`
- `fix(auth): validate JWT expiry`
- `build: upgrade vcpkg baseline`

## CI Pipeline

1. Install vcpkg dependencies
2. `cmake -B build -DCMAKE_BUILD_TYPE=Release`
3. `cmake --build build`
4. `cd build && ctest --output-on-failure`
5. Static analysis: `clang-tidy`

## Build

```bash
cmake -B build -DCMAKE_TOOLCHAIN_FILE=$VCPKG_ROOT/scripts/buildsystems/vcpkg.cmake
cmake --build build --config Release
```

## .gitignore

```
build/

```

## Docker

```dockerfile
FROM ubuntu:24.04 AS build
RUN apt-get update && apt-get install -y cmake g++ git curl zip
# Install vcpkg + deps
COPY . /app
WORKDIR /app
RUN cmake -B build -DCMAKE_BUILD_TYPE=Release && cmake --build build

FROM gcr.io/distroless/cc-debian12
COPY --from=build /app/build/my_function /my_function
CMD ["/my_function"]
```

## Tooling

- `compile_commands.json` — generated by CMake for IDE support.
- `clang-format` — auto-format source files.
- `clang-tidy` — static analysis.
